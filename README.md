# CRT Learning Repository  
This repository contains my foundational practice in **HTML**, **CSS**, and **JavaScript** as part of the Campus Recruitment Training (CRT) program, developed under the guidance of **Vivek Bhauser Sir**.

## Purpose  
To document core frontend concepts, strengthen technical fundamentals, and maintain a structured record of my learning progress.

## Focus Areas  
- **HTML:** Semantic structure and clean markup  
- **CSS:** Responsive design, layouts, and visual styling  
- **JavaScript:** DOM manipulation, events, and ES6+ basics

## Learning Outcomes  
- Improved understanding of web development fundamentals  
- Ability to build responsive and interactive interfaces  
- Adoption of clean, maintainable coding practices

## Usage  
Open the main HTML file in any modern browser to review the implemented concepts.
